# Leon &amp; Ilian – Düsseldorf-Wochenende

Brainstorm-Seite für unser gemeinsames Konzertwochenende am **10.–11. Juli 2026** in Düsseldorf.

Live unter: https://donaldvl.github.io/leon-ilian-geburtstag/

## Was ist das?

Eine kleine, statische HTML-Seite, auf der Leon und Ilian aus 13 Event-Vorschlägen ihre Favoriten markieren und Notizen hinterlassen können. Alle Eingaben werden lokal im Browser gespeichert (`localStorage`).

## Hosting

GitHub Pages, served from `main` branch, root folder. Eine leere `.nojekyll` Datei verhindert, dass Jekyll die Seite vor der Auslieferung anfasst.

## Lokale Vorschau

Einfach `index.html` im Browser öffnen — keine Build-Schritte, keine Abhängigkeiten.
